# Only macOS
## Compiling
- Compiling bot off
```shell
make
```
- Compiling bot on
```shell
make bonus
```
## Launch
- Maps 1
```shell
./so_long maps/simple.ber
```
- Maps 2
```shell
./so_long maps/simple2.ber
```
